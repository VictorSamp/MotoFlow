# 🏍️ MotoFlow

Web ERP System for Motorcycle Club Management

---

## 📌 About the Project

MotoFlow is an ERP system developed as a Final Year Project (TCC), aiming to centralize and automate the administrative and organizational management of motorcycle clubs.

The system replaces spreadsheets, messaging apps, and manual records with a centralized web platform.

---

## 🎯 Purpose

- Centralize member information
- Automate internal processes
- Improve event management
- Manage member progression
- Control internal responsibilities

---

## ⚙️ Tech Stack

- .NET 8 (ASP.NET Core)
- C#
- Entity Framework Core
- SQL Server
- Blazor Server
- Clean Architecture
- Docker

---

## 🏗️ Architecture

The system follows Clean Architecture principles:

- **Domain** → Business rules and entities
- **Application** → Use cases
- **Infrastructure** → Data persistence
- **Web/API** → Presentation layer

---

MVP in development with core member management features already implemented.
