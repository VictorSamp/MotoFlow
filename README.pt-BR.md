# 🏍️ MotoFlow

Sistema ERP Web para Gestão de Motoclubes

---

## 📌 Sobre o Projeto

O MotoFlow é um sistema ERP desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de centralizar e automatizar a gestão administrativa e organizacional de motoclubes.

O sistema substitui o uso de planilhas, aplicativos de mensagens e registros manuais por uma plataforma web centralizada.

---

## 🎯 Objetivo

- Centralizar informações de membros
- Automatizar processos internos
- Melhorar o controle organizacional de eventos
- Gerenciar progressão de membros
- Controlar responsabilidades internas da sede

---

## ⚙️ Tecnologias

- .NET 8 (ASP.NET Core)
- C#
- Entity Framework Core
- SQL Server
- Blazor Server
- Clean Architecture
- Docker

---

## 🏗️ Arquitetura

O sistema segue os princípios de Clean Architecture:

- **Domain** → Entidades e regras de negócio
- **Application** → Casos de uso (UseCases)
- **Infrastructure** → Persistência de dados
- **Web/API** → Interface e API

---
